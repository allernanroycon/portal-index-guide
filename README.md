# portal-index-guide

本仓库用于归档和发布多个独立 HTML 页面，主要作为静态资源索引与展示的集合点。仓库内容不针对任何特定域名或具体网站，仅提供通用型页面归档服务。

## 项目简介

portal-index-guide 是一个面向 HTML 页面的归档仓库，旨在整理、保存并发布各类独立前端页面。每个页面均可独立访问，不依赖后端服务或特定运行环境。仓库本身不包含任何指向具体网站或服务的链接，仅作为页面文件本身的管理与展示平台。

## 目录说明

```
portal-index-guide/
├── pages/          # 存放所有归档的 HTML 页面
├── assets/         # 页面引用的资源文件（CSS、JS、图片等）
├── index.html      # 仓库首页（可选，用于展示页面列表）
└── README.md       # 本文件
```

- **pages/**：每个子文件夹或独立文件对应一个完整的 HTML 页面，文件名应具有辨识度。
- **assets/**：按需组织资源，避免冗余。建议每个页面独立维护资源文件夹。
- **index.html**：可选自动生成页面索引，方便浏览。

## 页面归档说明

- 所有页面必须为纯静态 HTML，不包含外部服务调用或动态内容。
- 页面应自包含其样式与脚本，或引用仓库内资源。
- 命名规范：使用小写字母、数字和短横线，避免特殊字符。
- 每个页面可附带简短说明文档（可选），存放于页面同级目录下。

## 维护说明

- 提交新页面时，请确保文件完整且无外部依赖。
- 建议通过 Pull Request 提交修改，保持主分支稳定。
- 若需更新已有页面，请注明变更原因。
- 仓库维护者拥有对不合规页面的删除或修改权限。
